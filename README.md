# ResumeOS
Easily bootstrap an OS project to fool HR departments and pad your resume.

## Installation
```sh
pip install -r requirements.txt
```

Tested with python 2.7.10

## Usage
python resumeos.py --name 'My cool OS proejct' --output 'Hello world'
